# Projeto Pizzaria — Java

Este projeto demonstra o uso dos padrões Strategy e Decorator em um sistema de montagem de pizzas e formas de pagamento.

## Estrutura
- `pizza/` — classes base das pizzas
- `decorator/` — implementações de adicionais (padrão Decorator)
- `strategy/` — estratégias de pagamento
- `PizzariaApp.java` — ponto de entrada

## Como executar
Compile:
```
javac -d out src/**/*.java
```

Execute:
```
java -cp out PizzariaApp
```
